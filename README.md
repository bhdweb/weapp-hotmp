# 微信小程序-热门公众号文章

### 说明：

利用微信小程序官方开发者工具开发，使用截获的搜狗搜索GET请求包有关微信平台热门新闻与关键词的数据生成的新闻列表浏览。

### 数据接口:

使用本地数据

### 目录结构：

- pages — 存放项目页面相关文件，一共包括index,text等几个页面
- utils — 存放日期格式化文件

### 开发环境：

微信web开发者工具 v0.10.102800
